# Iris Species Predictor

**by Hamadullah Rajper**

---

I built this as a hands-on way to take a trained ML model off my laptop and put it in front of actual users through a browser. You type in four measurements, hit predict, and the app tells you which of the three Iris species you're looking at — along with how confident the model is.

The UI uses a dark forest-green and gold palette I designed specifically for this project. Nothing borrowed, nothing default.

---

## What it does

You provide the four standard Iris measurements:

- Sepal length and width
- Petal length and width

The Random Forest model (trained to 100% accuracy on the classic Iris dataset) returns one of three species — **Setosa**, **Versicolor**, or **Virginica** — along with a confidence percentage and a short description of the species.

---

## Getting it running locally

Make sure you have Python 3.8+ installed, then:

```bash
# 1. grab the code
git clone https://github.com/codebyhamad-create/iris-webapp.git
cd iris-webapp-hamadullah

# 2. install what's needed
pip install flask numpy scikit-learn

# 3. generate the model file (only needed once)
python train_model.py

# 4. start the server
python app.py
```

Then open `http://127.0.0.1:5000` in your browser.
